Flora Art Casa — 网站部署指南
================================

## 文件说明

index.html          → 网站主文件
data/products.json  → 产品数据（通过后台管理编辑）
data/settings.json  → 网站设置（邮箱、社交媒体等）
admin/index.html    → 后台管理入口
admin/config.yml    → 后台管理配置
netlify.toml        → Netlify 部署配置
images/uploads/     → 产品图片存放目录（后台上传的图片会存在这里）


## 部署步骤（按顺序操作）

### 第 1 步：注册 GitHub
1. 打开 https://github.com
2. 点击 Sign Up，用邮箱注册
3. 注册完登录

### 第 2 步：创建 GitHub 仓库
1. 点击右上角 "+" → "New repository"
2. 仓库名填：floraartcasa-website
3. 选 Public
4. 点 "Create repository"
5. 把本文件夹中的 **所有文件** 上传到仓库
   - 点 "uploading an existing file"
   - 把整个文件夹的内容拖进去
   - 点 "Commit changes"

### 第 3 步：注册 Netlify
1. 打开 https://app.netlify.com
2. 点 "Sign up" → 选 "GitHub" 登录
3. 授权 Netlify 访问你的 GitHub

### 第 4 步：部署网站
1. 在 Netlify 后台点 "Add new site" → "Import an existing project"
2. 选择 GitHub
3. 找到你刚才创建的 floraartcasa-website 仓库
4. 直接点 "Deploy site"
5. 等待 1-2 分钟，网站就上线了（会给你一个 xxx.netlify.app 临时网址）

### 第 5 步：绑定域名
1. 在 Netlify 后台 → Domain management → Add custom domain
2. 输入 floraartcasa.com
3. Netlify 会显示需要配置的 DNS 记录
4. 登录你的域名注册商后台
5. 把 DNS 解析修改为 Netlify 提供的地址
6. 等待生效（几分钟到几小时）
7. HTTPS 证书会自动配置

### 第 6 步：开启后台管理（Decap CMS）
1. 在 Netlify 后台 → Integrations → Identity → Enable Identity
2. 点 Identity 标签页 → Invite users → 输入你自己的邮箱
3. 去邮箱收邮件，点链接设置密码
4. 在 Netlify → Identity → Settings → Services → Enable Git Gateway
5. 现在访问 floraartcasa.com/admin 就可以登录后台了

### 第 7 步：配置 WhatsApp
1. 打开 index.html
2. 搜索 "wa.me/8613800138000"
3. 把 8613800138000 替换成你的真实 WhatsApp 号码（含国际区号）

### 第 8 步：配置 Tawk.to 在线客服（免费）
1. 打开 https://www.tawk.to
2. 注册账号（免费）
3. 创建一个 Property（输入你的网站域名）
4. 复制它给你的嵌入代码
5. 打开 index.html，搜索 "TAWK.TO LIVE CHAT"
6. 用你的嵌入代码替换那段注释


## 后台使用说明

登录 floraartcasa.com/admin 后可以：

### 管理产品
- 左侧点"产品管理" → "产品列表"
- 点任何产品可以编辑名称、分类、描述、图片等
- 点右上角列表底部的 "Add" 可以添加新产品
- 编辑完点 "Publish" 发布，网站会自动更新

### 修改网站设置
- 左侧点"网站设置" → "基本信息"
- 可以修改邮箱、WhatsApp 号码、社交媒体链接等


## 常见问题

Q: 后台登录不了？
A: 确认已在 Netlify → Identity 中邀请了你的邮箱，并开启了 Git Gateway。

Q: 产品图片怎么上传？
A: 在后台编辑产品时，点"产品图片"字段的上传按钮，选择图片即可。图片会自动上传到 GitHub 仓库的 images/uploads 目录。

Q: 修改后网站没更新？
A: 后台发布后，Netlify 需要 1-2 分钟重新部署。刷新页面或清除浏览器缓存再看。

Q: 询盘表单提交后我怎么收到？
A: 在 Netlify 后台 → Forms 页面可以看到所有提交。也可以在 Forms → Notifications 设置邮件通知，每次有客户提交都发邮件给你。
